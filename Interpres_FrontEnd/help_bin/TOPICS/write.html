Saving Files

  1. Click the "File" button on the menu strip.
  2. Click the "Save As" button in the File menu.
  3. Select an existing file, or create a new file, using the save dialog.
    • By default, only .txt files are shown. You can change this by
      clicking the dropdown in the bottom right, and changing it
      from "Text Files" to "All Files".
    • You can also enter a custom file extension, when the filter is set
      to "All Files".
  4. Click Save button to save the file.
